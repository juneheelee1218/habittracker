One-time setup 🔧

To sync between Junehee & Carolina's devices, you need a free Firebase Realtime Database. Takes ~3 minutes:

  1. Go to console.firebase.google.com
  2. Click Add project → name it anything → click through
  3. Click Realtime Database in the left sidebar → Create database → choose Start in test mode
  4. Go to ⚙️ Project settings → scroll to Your apps → click </> (Web) → register app
  5. Copy the firebaseConfig object and paste it below
